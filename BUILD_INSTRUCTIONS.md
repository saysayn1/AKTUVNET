# Инструкция по сборке AKTUVNET в EXE

## Подготовка

### 1. Установка зависимостей
Откройте PowerShell в папке проекта и выполните:
```powershell
npm install
```

### 2. Добавление иконки приложения
Поместите файл `icon.png` (размер 512x512 или 1024x1024) в корень проекта.
Это будет иконка вашего приложения.

### 3. Добавление звуковых файлов
Поместите следующие файлы в папку `sounds/`:
- `message.mp3` - звук нового сообщения
- `incoming-call.mp3` - звук входящего звонка
- `call-connected.mp3` - звук подключения к звонку
- `friend-request.mp3` - звук запроса в друзья

## Сборка приложения

### Для Windows (EXE):
```powershell
npm run build-win
```

Готовый установщик будет в папке `dist/`

### Для тестирования в режиме разработки:
```powershell
npm run electron
```

## Результат

После сборки в папке `dist/` появится:
- **AKTUVNET Setup 1.0.0.exe** - установщик приложения

## Установка приложения

1. Запустите `AKTUVNET Setup 1.0.0.exe`
2. Выберите папку для установки
3. Приложение создаст ярлык на рабочем столе и в меню Пуск
4. Запустите AKTUVNET!

## Особенности

- Приложение автоматически запускает сервер при старте
- Не требует отдельного запуска Node.js сервера
- Работает как обычное Windows приложение
- Все данные хранятся локально в SQLite базе данных

## Требования к иконке

Для лучшего результата создайте иконку:
- Формат: PNG
- Размер: 512x512 или 1024x1024 пикселей
- Прозрачный фон (опционально)
- Яркий и узнаваемый дизайн

Можете использовать онлайн-редакторы:
- Canva.com
- Photopea.com
- Figma.com

## Решение проблем

### Ошибка "electron-builder not found"
```powershell
npm install --save-dev electron-builder
```

### Ошибка при сборке
Убедитесь что:
1. Установлен Node.js версии 16 или выше
2. Все зависимости установлены (`npm install`)
3. Файл `icon.png` существует в корне проекта

### Приложение не запускается
Проверьте что:
1. Порт 3000 свободен
2. Нет запущенных копий приложения
3. Антивирус не блокирует приложение
