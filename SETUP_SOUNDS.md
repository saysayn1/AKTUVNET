# Настройка звуков для AKTUVNET

## Ваши звуковые файлы

У вас есть следующие звуки:
- звонок
- отключился
- подключен
- поп.mp3
- сообщение.wav ⭐ (для запросов в друзья)

## Шаги по настройке:

### 1. Переименуйте файлы

Переименуйте ваши звуковые файлы следующим образом:

```
поп.mp3 → message.mp3
звонок → incoming-call.mp3
подключен → call-connected.mp3
сообщение.wav → friend-request.wav ⭐ (отдельный звук для друзей!)
```

### 2. Опционально: завершение звонка

Если хотите звук при завершении звонка:
```
отключился → call-ended.mp3
```

### 3. Переместите файлы

Поместите все 4 файла в папку:
```
C:/Users/hplap/OneDrive/Desktop/AKTUVNET/sounds/
```

### Итоговая структура:

```
AKTUVNET/
└── sounds/
    ├── message.mp3          (из поп.mp3) - звук сообщения
    ├── incoming-call.mp3    (из звонок) - входящий звонок
    ├── call-connected.mp3   (из подключен) - подключение
    ├── friend-request.wav   (из сообщение.wav) - запрос в друзья ⭐
    └── call-ended.mp3       (из отключился) - завершение звонка (опционально)
```

## Дополнительные звуки (опционально)

Если хотите использовать:
- **отключился** - можно добавить как `call-ended.mp3` для завершения звонка
- **сообщение.wav** - конвертируйте в MP3 если хотите использовать вместо поп.mp3

## Конвертация WAV в MP3

Если нужно конвертировать WAV в MP3:
1. Используйте онлайн конвертер: https://convertio.co/ru/wav-mp3/
2. Или используйте VLC Media Player: Медиа → Конвертировать/Сохранить

## Проверка

После настройки запустите приложение:
```powershell
npm start
```

Звуки должны проигрываться при:
- Получении нового сообщения
- Входящем звонке (зацикленный)
- Подключении к звонку
- Получении запроса в друзья

## Настройка громкости

Если звуки слишком громкие/тихие, отредактируйте `sounds.js`:

```javascript
// Для сообщений
audio.volume = 0.5;  // Измените от 0.0 до 1.0

// Для входящего звонка
this.incomingCallAudio.volume = 0.6;  // Измените от 0.0 до 1.0
```
