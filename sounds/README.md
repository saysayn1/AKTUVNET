# Звуковые файлы для AKTUVNET

## Необходимые звуковые файлы:

Поместите следующие звуковые файлы в эту папку:

1. **message.mp3** - Звук при получении нового сообщения
   - Рекомендуемая длительность: 0.5-1 секунда
   - Формат: MP3
   - Пример: короткий "пинг" или "бип"

2. **incoming-call.mp3** - Звук входящего звонка (зацикленный)
   - Рекомендуемая длительность: 2-3 секунды (будет повторяться)
   - Формат: MP3
   - Пример: мелодия звонка телефона

3. **call-connected.mp3** - Звук при подключении к звонку
   - Рекомендуемая длительность: 0.5-1 секунда
   - Формат: MP3
   - Пример: короткий "бип" подтверждения

4. **friend-request.wav** или **friend-request.mp3** - Звук при получении запроса в друзья
   - Рекомендуемая длительность: 0.5-1 секунда
   - Формат: WAV или MP3 (оба поддерживаются)
   - Пример: приятный уведомительный звук

5. **call-ended.mp3** (опционально) - Звук при завершении звонка
   - Рекомендуемая длительность: 0.5-1 секунда
   - Формат: MP3
   - Пример: короткий "бип" отключения

## Где скачать звуки:

- **Freesound.org** - бесплатные звуки
- **Zapsplat.com** - бесплатные звуковые эффекты
- **Mixkit.co** - бесплатные звуки для проектов

## Примеры поисковых запросов:

- "notification sound"
- "message ping"
- "phone ringtone"
- "call connected beep"
- "friend request notification"

После добавления звуковых файлов в эту папку, они автоматически будут использоваться приложением.
